# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1, as its Code of Conduct.

The full text is available at
[contributor-covenant.org/version/2/1/code_of_conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

To report a concern about participant conduct, contact the repository
maintainer through their [GitHub profile](https://github.com/arminfabritzek).

Reports will be reviewed and handled confidentially.
