# Contributing to WatchMyBirds

Thank you for your interest in contributing to WatchMyBirds.

This top-level file exists so GitHub can detect contribution guidance for the
repository. The detailed contributor guide lives here:

- [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md)

## Quick Checklist

- Open an issue before starting large features or structural refactors.
- Keep pull requests focused and easy to review.
- Add or update tests when changing core behavior.
- Follow the existing project style and commit conventions.

## Development Notes

- Main project documentation: [README.md](README.md)
- Architecture and invariants: [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md),
  [docs/INVARIANTS.md](docs/INVARIANTS.md)
- Security reporting: [SECURITY.md](SECURITY.md)

If you want the full setup, coding-style, and PR guidance, use the detailed
document in [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md).
