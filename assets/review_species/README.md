Review species images for `/admin/review`.

Naming:
- Use the scientific species key from the app UI.
- Format: `<Scientific_Name>.<ext>`
- Example: `Cyanistes_caeruleus.jpg`

Supported extensions, in lookup order:
- `webp`
- `jpg`
- `jpeg`
- `png`
- `avif`

Folder URL base:
- `/assets/review_species/`
