# Optional dependencies for the nightly aesthetic tagger
# (web/services/aesthetic_tag_scheduler.py + scripts/aesthetic_tag_nightly.py).
#
# Kept separate from the main requirements.txt because:
#  * torch + torchvision pull ~900 MB, which doubles the Docker image size
#    and slows cold-pull. Some deployments may not need the tagger at all.
#  * The CPU-only PyTorch wheels (~150 MB instead of ~1.5 GB with cuDNN)
#    require a custom index URL, which would clutter the main file.
#
# Install with:
#   pip install -r requirements-aesthetic.txt
#
# The index directives below are pinned in-file so a plain `-r` install
# pulls torch/torchvision from PyTorch's CPU distribution. Without them,
# a bare `pip install torchvision>=0.27` resolves to the PyPI CUDA wheels
# and drags ~3.8 GB of nvidia-*/cuda-*/triton onto CPU-only hosts (e.g.
# the Raspberry Pi appliance). --extra-index-url falls back to PyPI for
# open_clip_torch.

--index-url https://download.pytorch.org/whl/cpu
--extra-index-url https://pypi.org/simple

torch>=2.0
torchvision>=0.27.0
open_clip_torch>=3.3.0
